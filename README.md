# AI-and-LLMs

to run any of the model , make sure to clone the repo and use the following commands on the vscode or any editor terminal
1- git clone https://github.com/SHASHANKKUSHWAHA777/AI-and-LLMs.git
go to the folder using cd
2- python -m venv .venv
.venv\Scripts\activate
make sure to activate the virtual environment , everything will run inside the virtual environment
now , install the dependencies
3- pip install -r requirements.txt
4- install ollama from browser
after installing ollama , run the following command (ollama run modelname)
5- ollama pull llama3 // ollama pull phi3 
and finaly you are ready to run the model
6- streamlit run filename.py
